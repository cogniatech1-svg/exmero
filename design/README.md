# design/

Este directorio centralizará la identidad visual oficial de ExMero cuando la biblioteca de marca quede congelada.

## Estado actual

> **En construcción.** La biblioteca oficial de Mero está siendo desarrollada.
> Ningún recurso dentro de este directorio debe considerarse canónico hasta que exista confirmación explícita.

## Estructura prevista

```
design/
  source_of_truth/          ← Todo lo que esté aquí es canónico
    logo/                   ← Logotipo en sus variantes oficiales
    mero/
      emociones/            ← Expresiones faciales
      gestos/               ← Poses completas del cuerpo
      estudio/              ← Mero en contextos de aprendizaje
      simulacros/           ← Mero en contextos de práctica
      estados/              ← Estados especiales (racha, logro, etc.)
      transparentes/        ← Versiones PNG con fondo transparente
    ilustraciones/          ← Ilustraciones de escena y contexto
    iconos/                 ← Sistema de íconos lineal personalizado
    colores/                ← Paleta oficial exportada (tokens, Figma, etc.)
    tipografia/             ← Archivos de fuentes y guías de uso
```

## Referencia

Ver `docs/marca/BRAND_SOURCE_OF_TRUTH.md` para las reglas completas de uso de la marca
y la lista de recursos descartados que NO deben usarse como referencia.
